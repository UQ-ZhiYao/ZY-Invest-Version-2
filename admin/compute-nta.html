How Daily Fee Accrual Works

Base Fee (daily)
Base Fee = AUM × rate% / 365

Accrues every calendar day regardless of performance. Deducted from NTA daily.

Performance Fee (daily)
Annualised = NTA^(365 / days_since_inception) − 1
If Annualised > Hurdle Rate:
   Excess   = Annualised − Hurdle Rate
   Daily Fee = Units × (Excess ÷ 365) × Rate

Accrues daily only when the fund's annualised return since inception exceeds the hurdle. Inception NTA = 1.0000.
